Allow Login as Any User

Allow Login as Any User is a flexible application which provides change of user from Admin to any User or from any User to any User without password. And to login as Admin from any User, password is required for security reasons.

With Ksolves Allow Login as Any User facilitate you with easy switching between Users.


Hot Features

  • Login Access : Any user will be able to login to any other user’s account without password, by just clicking on the Login button. However, to login as admin, the user has to give the password
  • Admin Access : The admin has an option to login to any user’s account, however, the reverse is not possible
  • Mark as Active : Any user who is currently logged in, will be marked as active in the module
