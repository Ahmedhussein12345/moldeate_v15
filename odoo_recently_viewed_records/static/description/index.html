Odoo Recently Viewed Records

Track your recently viewed records and work without worry.

Odoo recently viewed records to help to keep track of your activity (up to 10) and help you edit the records if required. With just a click, you can track, record, and view your history and the admin can view other users' history as well. Further, the system maintains a historical record that can be viewed later on.

Keep recently viewed records of all your latest activity

You were working on a project then suddenly your manager asked you to switch your task because of a priority project that requires your immediate attention. You stopped the work and started the new project and completed it.

But you forgot which task you were doing and where you were. Odoo recent viewed records help you to access the task you were doing with just a click.

Recently viewed records are maintained even after logging out

If you are worried about your session history after logging out then relax. The system will maintain login and log-out history, the module will maintain the history even after the session was logged out. And users will be able to see the history when they log in again through the recent history icon.

Can only be accessed by admin

Safety and security is major concern for anybody. So to maintain it, Recently viewed records of both user and admin can be viewed by an admin. And the user can only see their own history so the data is secure and safe. No one can access or view it apart from the user or admin.

Maintain multiple user data

Admin can view every user recently viewed records through the user section and show them choices as per their preferences. Also, they can maintain the history data separately in a file format in their system or server.
